#### expo-cli 설치

```
npm install --global expo-cli
```

#### 프로젝트 생성

```
expo init 프로젝트명
> blank (TypeScript) 선택해서 진행함.
```

#### 프로젝트 실행

```
npm run start

# 또는

npx expo start # 그냥 expo start를 하면, 버전 확인하라는 문구가 발생하는 듯. (npx expo 랑 그냥 expo와 버전차이가 있는 듯함.)
```
